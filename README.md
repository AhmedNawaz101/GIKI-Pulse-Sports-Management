# GIKI Pulse Sports Management

## Project Name
GIKI Pulse Sports Management

## Overview
Sports Complex Booking Suite delivers a polished web experience for managing court reservations, team coordination, event signups, and fee payments. Designed for student-athletes and facility admins, it streamlines daily workflows across booking, announcements, trials, payments, and team management in one responsive portal. With intuitive controls.

## Technologies Used
- HTML
- CSS
- JavaScript

## Key Features
- Booking workflow for personal and society reservations
- Event browsing and trial registration
- Payment input and confirmation flow
- Team management and coach request simulation
- Announcement and activity pages
- Responsive front-end navigation and drawer menus

## Project Structure
- `index.html` - Welcome screen
- `signin.html` - Sign-in page
- `home.html` - Main dashboard and quick actions
- `booking.html` - Booking interface
- `events.html` - Events listing and application flow
- `payment_input.html` - Payment details form
- `confirmation_booking.html` / `confirmation_trial.html` - Confirmation pages
- `team_management.html` - Team access and coach preview
- `announcements.html` - Announcements and inbox
- `activity.html` - Activity page
- `style.css` / page-specific CSS files - styles
- `app.js` - client-side navigation and UI behavior

## Getting Started
1. Open the project folder in your code editor.
2. Open `index.html` in your browser.
3. Use the navigation buttons and quick actions to explore the app.

## Notes
